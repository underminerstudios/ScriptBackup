NormalMap Checklist
author: Norman Schaar (norman.schaar@gameloft.com)
  • Gamma: Gamma needs to be turned off to ensure proper render and display of normal maps. Normal maps work in linear space.
  • Qualified Normals: The tangent basis calculation differs between the 3dsmax renderer and 3dsmax viewport. Turned qualified on syncs both calculations.
  • True Color: True Color needs to be off. This feature is from the DOS days when 16bit images could not be saved only 8-bit images. True Color adds noise to the textures rendered in 3dsmax to avoid banding. This is not needed these days.
