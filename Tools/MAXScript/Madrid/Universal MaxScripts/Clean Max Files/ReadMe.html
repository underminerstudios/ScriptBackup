Clean Max Files
author: Norman Schaar (norman.schaar@gameloft.com)
  • This script will batch-process all max files in the root and subfolders of the specified folder.
  • You have the option to delete unusued materials in the max scene, this will get rid of unnecessary references.
  • This script will attempt to fix paths of any resource which is not found by looking inside the folders and sub-folders of each max file. Additional paths can be provided to increase the chances of finding the missing resources.
