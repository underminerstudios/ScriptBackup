Pivot Tools
author: Norman Schaar - norman.schaar@gameloft.com
  • Vert Snap: It will snap the pivot point to the currently selected vert. If multiple are selected the pivot will be snapped to the center of the selection.
  • Knot Snap: It will snap the pivot point to the currently selected knot. If multiple are selected the pivot will be snapped to the center of the selection.
  • Center: It will snap the pivot point to the center of each selected object.
  • Bottom: It will snap the pivot point to the bottom of each selected object.
