Sort Sub-Element Transparency
author: Norman Schaar (norman.schaar@gameloft.com)
  • This script changes the order of the faces in an object so that they are rendered properly. Select a dummy and select as many meshes as you want.
  • The script will calculate the distance between the dummy and the mesh elements and will change the order so that the elements that are further away from the dummy have the lowest ID number possible. Glitch will render them in this order.
