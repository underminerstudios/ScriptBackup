Assign Lightmaps
author: Norman Schaar - norman.schaar@gameloft.com
  • This script will allow you to add lightmaps to multiple DirectX9 materials at the same time. These DirectX materials need a LightMap slot.
  • Use "Select" to select a list of lightmaps.
  • Use the drop down list to select one of these.
  • Click "Assign" to add this lightmap to the materials of the current selected objects.
