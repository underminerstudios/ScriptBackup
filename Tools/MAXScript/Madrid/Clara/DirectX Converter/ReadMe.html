DirectX Converter
author: Norman Schaar (norman.schaar@gameloft.com)
  • This script will allow you to change a DirectX Material to a Standard material and vice versa. The DirectX Shader needs to have the following properties:
    • DiffuseMap
    • LightMap
    • SpecularMap
    • NormalMap
  • They will be converted and moved to the following slots:
    • DiffuseMap
    • AmbientMap
    • SpecularLevel
    • BumpMap
