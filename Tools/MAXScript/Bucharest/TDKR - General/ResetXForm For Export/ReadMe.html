ResetXForm For Export
author: Gabriel Coriiu (gabriel.coriiu@gameloft.com)
  • Checks if two or more objects have the same name
  • Checks if one or more objects have bad faces (surface ~= 0) and select them if any
  • Checks if one or more objects have invalid LOD names
    • *_LODD
    • *_LOD00, *_LOD01, *_LOD02,... *_LOD09
    • *_LOD10, *_LODD01, *_LOD1 etc.
  • Removes all map channels > 2 and all map channels (-2 to 99) if matchPattern obj.name "*_Collision" (e.g. Footprint_Collision will have no map channel since is supposed to be collision)
    • NOTE: Footprint_Collision_00 is NOT treated as collision
