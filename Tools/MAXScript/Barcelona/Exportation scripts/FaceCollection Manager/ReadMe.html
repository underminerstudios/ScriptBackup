Face Collection Manager
author: Marc Marti - marc.marti@gameloft.com
DESCRIPTION:
  • This script will allow you to attach a bunch of objects change their mapping coordinates and detach them gain preserving their names and properties.
  • This is specially usefull, as instance, to bake a single lightmap from a bunch of separated objects that belongs to the same sector and use the same lightmap but they should be exported separatedly.
  • IMPORTANT: Save your scene before you start using this script!
INSTRUCTIONS:
  • Select the desired objects to attach.
  • Press [Attach selected objects].
  • Change their mapping coordinates.
  • IMPORTANT: Do not weld or delete any vertex or face. If you change any geometry elements the script will fail and you will not be able to dettach the meshes!
  • Press [Detach all collections].
NOTES:
  • This script deals with quite delicate processes and can mess the things a lot if you don't use it apropiately, so, please be sure that you SAVE YOUR SCENE BEFORE YOU START USING THIS SCRIPT.
  • If you try to close the script an alert window will appear. This is totally normal, and it's to prevent closing the window by mistake, leaving all the objects attached.
