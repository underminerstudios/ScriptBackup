Object orientator
author: Marc Marti - marc.marti@gameloft.com
DESCRIPTION:
  • This script will allow you to orient the selected objects pointing a previously selected object.
  • The selected objest will be pointing their negative Z axis towards the target object.
INSTRUCTIONS:
  • Select the the target object. (All the objects will look at this object)
  • Select all the objects you want to reorient.
  • Click [Orientantor] button.
KNOWN ISSUES:
  • Due the script is pointing the selected objects toewards the target object using the negative Z axis, may happen that the final orientation of those objects is noolt really pointing the target. To re-orient the objects use the local transformation pivot center to rotate all the objects locally and correct their final orientation.
