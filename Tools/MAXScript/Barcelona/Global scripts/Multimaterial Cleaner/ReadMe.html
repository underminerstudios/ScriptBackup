Multimaterial Cleaner
author: Marc Marti - marc.marti@gameloft.com
DESCRIPTION:
  • This script will allow you to clean duplicated materials and badly-sorted ID's inside all the selected objects multimaterials.
INSTRUCTIONS:
  • Select the objects that you want to clean.
  • Click the [Clean Multi. Material] button.
KNOWN ISSUES:
  • Due the script is mergin all the materials that has a common diffuse name, ocasionally, this could cause some artifact merging in a single material two materials with the same diffuse. THe easiest solution to avoid this kind of artifacts is to dettach one of those materials previously to run the script.
