Object follows FaceCenter
author: Marc Marti - marc.marti@gameloft.com
DESCRIPTION:
  • This script will allow you to position copies or instances along the face-centers of another object used as reference grid.
  • This is specially usefull if you want to position some objects in the face centers of a track, following the flow of the track object.
  • You can use this script with any kind of object.
INSTRUCTIONS:
  • Choose the grid object with the "pick" button.
  • Select the object to be copied or isntanced along the faces.
  • Click the [Obj to Face-centers] button.
