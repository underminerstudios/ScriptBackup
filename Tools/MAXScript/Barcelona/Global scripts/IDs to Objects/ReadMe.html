ID to objects splitter
author: Marc Marti - marc.marti@gameloft.com
DESCRIPTION:
  • Split an object by its IDs creating new singled-material objects.
INSTRUCTIONS:
  • Double click the script name. The script works directly from the menu.
KNOWN ISSUES:
  • Depending on the Sub-Materials inside the Multisubobject material of the object, the script may detach the sub-objects in a weird way.
