Object replacer
author: Marc Marti - marc.marti@gameloft.com
DESCRIPTION:
  • This script will allow you to replace, re-orient and re-scale the selected objects.
  • You can replace the selected objects with copys or instances of the "master object".
  • You can use this script with any kind of object.
INSTRUCTIONS:
  • Slelect the desired clonation method (copy or instancing).
  • Choose the randomization methods that you wnat to apply over the new created objects (rotation and scale).
  • Select the master object. (All the objects will be instances of copys of this object)
  • Click the [Replace] button.
