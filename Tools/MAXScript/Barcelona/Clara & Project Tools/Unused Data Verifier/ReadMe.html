Unused Data Verifier
author: Marc Marti - marc.marti@gameloft.com
DESCRIPTION:
  • This script will allow you to check for potentially unused data assets on an entire project folder. This means identify all the unused DAE files and TGA files inside a project.
  • VERY IMPORTANT: Due the nature of the script and the amount of assets in some projects, it can take A LOT of time to parse an entire project. I recommend you to run the script when you are not busy with your computer (as sintance before you leave the office) Be patient, finally the list will appear.
INSTRUCTIONS:
  • Set the [Gfx3D] folder of the project.
  • Set the [Levels] folder of the project.
  • Set the [States] folder of the project.
  • Press [search unused assets].
KNOWN ISSUES:
  • After deleteing the neecesary unused DAE files, it si recomendable to run the script again. Probably the script will be able too find more unused textures (those textures that were used only by those deleted DAE files.)
